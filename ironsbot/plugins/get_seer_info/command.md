# 赛尔号数据查询插件

赛尔号数据查询插件。

- 支持**名称模糊搜索**和 **ID 精确查找**；多个结果时会提示选择。
- 名称搜索时可省略 `·` `-` 等符号，不区分大小写。
- 所有命令需放置在消息**开头或结尾**才能触发，命令和参数间可不添加空格。

---

## 精灵相关

### 精灵信息

查询精灵基础信息（属性、魂印、技能等）。

| 命令   | 别名                           | 参数        |
| ------ | ------------------------------ | ----------- |
| `精灵` | `查询精灵信息`、`魂印`、`技能` | `<名称/ID>` |

**示例：**

```
精灵雷伊        # 查询名称含"雷伊"的精灵信息
奥菲魂印        # 通过别名"魂印"查询奥菲的信息
圣武技能        # 通过别名"技能"查询圣武的信息
```

---

### 精灵立绘

查询精灵或皮肤立绘图片。

| 命令   | 别名       | 参数        |
| ------ | ---------- | ----------- |
| `立绘` | `查询立绘` | `<名称/ID>` |

**示例：**

```
雷伊立绘        # 查询雷伊的立绘
立绘 1          # 查询 ID 为 1 的精灵立绘
```

---

## 刻印相关

### 刻印信息

查询刻印信息及数值，同时支持**刻印系列名称**搜索。

| 命令   | 别名 | 参数        |
| ------ | ---- | ----------- |
| `刻印` | 无   | `<名称/ID>` |

**示例：**

```
刻印v8          # 查询名称含"v8"的刻印
刻印精灵王       # 查询"精灵王"系列刻印
```
